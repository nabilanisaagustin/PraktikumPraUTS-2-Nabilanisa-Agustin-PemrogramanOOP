# Praktikum Pemrograman Berorientasi Objek (OOP) - Java

Repositori ini berisi kumpulan tugas praktikum pra UTS(2) Java.

## Daftar Isi Praktikum
Berikut adalah rincian materi yang telah diimplementasikan:

### 1. Method & Parameter (Bagian G)
* **G2_Tugas2**: Class Matematika dengan parameter (Tambah, Kurang, Kali, Bagi).
* **G3_Tugas3**: Class MobilLengkap (Pemanggilan method sederhana).
* **G4_Praktik7**: Implementasi **Polimorfisme** pada objek Hewan.

### 2. Overriding & Overloading (Bagian H)
* **H1_Overriding**: Menimpa method gambar() pada Class Bentuk (Segitiga & Persegi).
* **H2_Overloading**: Variasi parameter pada method tambah() di Class Hitung.

### 3. Abstraksi / Abstract Class (Bagian I)
* **I1_Abstraksi**: Penerapan `abstract class` pada MakhlukHidup (Manusia & Hewan).
* **I2_Abstraksi**: Penerapan `abstract class` pada Kendaraan (Mobil & Sepeda).

---

## Cara Menjalankan
1. Masuk ke folder tugas (contoh: `cd G2_Tugas2`).
2. Kompilasi kode: `javac *.java`
3. Jalankan program: `java [NamaFileUtama]`
